# IP address :
•	An IP address is a unique identifier assigned to a device on a network. It allows devices to communicate with each other.
•	There are mainly two types of ip address ipv4 and ipv6

# 1.	IPv4:
-	An IPv4 address consists of four octets separated by dots. Each octet contains 8 bits, resulting in a total address length of 32 bits.
-	It is a 32 bit numeric address
-	Each octet can have a value from 0 to 255.
-	A device can only process binary data (0s and 1s), so an IPv4 address is internally represented and processed as a binary number.
  
-	Below is a classical example of the conversion.
IPv4 Octet Conversion Example.
Example IPv4 Address: 192.168.1.10

Octet Chart
128 | 64 | 32 | 16 | 8 | 4 | 2 | 1

Convert 192 :
192 = 128 + 64
Binary: 11000000

Convert 168 :
168 = 128 + 32 + 8
Binary: 10101000

Convert 1 :
1 = 1
Binary: 00000001

Convert 10 :
10 = 8 + 2
Binary: 00001010


Final Result :
192.168.1.10 = 11000000.10101000.00000001.00001010

// This will be the address read by computer
